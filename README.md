# Luxe Boutique - Fashion Website

A modern, responsive fashion boutique website featuring elegant design, product catalog, and seamless shopping experience.

## Features

- **Responsive Design**: Mobile-first design that works perfectly on all devices
- **Product Catalog**: Advanced filtering, search, and sorting functionality
- **Shopping Cart**: Interactive cart with local storage persistence
- **Modern UI/UX**: Clean, elegant design with smooth animations
- **Quick View Modal**: Detailed product previews without page navigation
- **About Us Page**: Brand story, team information, and store gallery
- **Contact Integration**: Contact forms and social media links

## Pages

- `index.html` - Homepage with hero section and featured products
- `products.html` - Product catalog with filtering and search
- `about.html` - About us page with brand story and team
- `styles.css` - Complete responsive styling
- `script.js` - Interactive functionality and shopping cart

## Technologies Used

- **HTML5**: Semantic markup with modern standards
- **CSS3**: Flexbox, Grid, animations, and responsive design
- **JavaScript**: ES6+, local storage, and DOM manipulation
- **Font Awesome**: Icon library for UI elements
- **Google Fonts**: Playfair Display and Inter font families

## Getting Started

1. Open `index.html` in your web browser, or
2. Serve the files using a local web server:
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve .

   # Using PHP
   php -S localhost:8000
   ```

3. Navigate to `http://localhost:8000` in your browser
